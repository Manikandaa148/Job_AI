## Slide 7: Technical Architecture

**Modern, Scalable Stack:**
*   **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS.
*   **Backend:** FastAPI (Python), SQLAlchemy ORM.
*   **Database:** SQLite (Dev) / PostgreSQL (Prod) with SQLAlchemy.
*   **Security:** JWT Authentication, Pydantic Data Validation, BCRYPT hashing.
*   **AI/Logic:** Custom Python agents for profile matching and validation.

**Speaker Notes:**
We chose a best-in-class stack. Next.js ensures a fast, SEO-friendly frontend, while FastAPI provides high-performance asynchronous handling for our AI agents and heavy logic.
